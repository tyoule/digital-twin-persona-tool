Twin-2K-500 — Persona Browser

Select a persona, build a prompt, and copy it for your LLM experiments.

This tool provides an interface for the Twin-2K-500 dataset, developed by Toubia, Gui, Peng, and John (2025).

Experimental Workflow

  1. Select Subject: Load a digital twin profile using "Random Persona" or a specific ID number.
  2. Design Experiment: Customize the Scenario (treatment scenario or control scenario) and Question (outcome) below to test your hypothesis.
  3. Run Simulation: Click "Generate Prompt," copy the text, and paste it into your LLM of choice to observe the response.
